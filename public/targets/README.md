# targets/

Place your compiled `campus.mind` file here.

Generate it at: https://hiukim.github.io/mind-ar-js-doc/tools/compile

Steps:
1. Take a clear photo of your laser-cut campus model
2. Upload to the compiler linked above
3. Download the compiled file
4. Rename to: campus.mind
5. Place it in this folder: public/targets/campus.mind
